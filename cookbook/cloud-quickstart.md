## 🚀 快速开始

### 1. 注册登录

**步骤**：
1. 访问 https://app.all-hands.dev
2. 点击 "Sign in with GitHub" 或 "Sign in with GitLab"
3. 授权登录
4. 完成！

### 2. 免费试用

**免费额度**：
- ✅ **Minimax 模型** - 免费使用
- ✅ **基础功能** - 完全免费
- ✅ **无限制会话** - 随时使用

### 3. 创建第一个会话

```bash
# 步骤：
1. 点击 "New Conversation"
2. 输入任务描述
3. 选择 Minimax 模型（免费）
4. 点击 "Start"
```

## 🎯 基础使用

### 1. 界面介绍

**主界面**：
- **左侧**：会话列表
- **中间**：对话区域
- **右侧**：操作日志

**顶部工具栏**：
- **模型选择**：切换不同模型
- **设置**：配置参数
- **分享**：分享会话

### 2. 任务示例

#### 示例 1：创建 Web 项目

```
输入：
帮我创建一个 FastAPI 项目，包含：
1. 用户注册/登录
2. JWT 认证
3. SQLite 数据库
4. 单元测试

Agent 自动执行：
✅ 创建项目结构
✅ 编写代码
✅ 配置数据库
✅ 编写测试
```

#### 示例 2：代码优化

```
输入：
优化这段 Python 代码的性能：
[粘贴代码]

Agent 自动执行：
✅ 分析性能瓶颈
✅ 提供优化建议
✅ 重构代码
✅ 对比性能差异
```

#### 示例 3：Bug 修复

```
输入：
修复这个错误：
RuntimeError: CUDA out of memory

代码：
[粘贴代码]

Agent 自动执行：
✅ 诊断问题
✅ 提供解决方案
✅ 修复代码
✅ 添加错误处理
```

### 3. 会话管理

**保存会话**：
- 点击右上角 ⋮ → Save
- 输入会话名称
- 自动保存到云端

**导出会话**：
- 点击 ⋮ → Export
- 选择格式（JSON/Markdown）
- 下载到本地

**分享会话**：
- 点击 ⋮ → Share
- 生成分享链接
- 发送给他人

## 🔧 高级功能

### 1. 集成 GitHub/GitLab

**连接仓库**：
1. Settings → Integrations
2. 连接 GitHub/GitLab
3. 授权访问
4. 选择仓库

**自动同步**：
- 代码修改自动提交
- 创建 Pull Request
- 触发 CI/CD

### 2. 团队协作

**创建团队**：
1. Settings → Team
2. 邀请成员
3. 设置权限
4. 共享会话

**协作功能**：
- ✅ 实时协作
- ✅ 评论讨论
- ✅ 版本历史
- ✅ 权限管理

### 3. Slack 集成

**配置 Slack**：
1. Settings → Integrations → Slack
2. 授权 Slack 工作区
3. 选择频道
4. 配置通知

**使用方式**：
```bash
# 在 Slack 中使用
/openhands create FastAPI 项目
/openhands list
/openhands show <session-id>
```

### 4. Jira/Linear 集成

**连接 Jira**：
1. Settings → Integrations → Jira
2. 输入 Jira URL
3. 输入 API Token
4. 选择项目

**自动创建任务**：
- Agent 完成任务后
- 自动更新 Jira 状态
- 添加评论和链接

## 🎓 最佳实践

### 1. 任务描述技巧

❌ **不好的描述**：
```
优化代码
```

✅ **好的描述**：
```
优化 main.py 中的数据处理函数：
1. 减少内存占用（目标：<100MB）
2. 提高处理速度（目标：<1秒）
3. 保持代码可读性
4. 添加单元测试
```

### 2. 模型选择建议

| 模型 | 推荐场景 | 成本 |
|------|---------|------|
| **Minimax** | 免费试用、快速验证 | Free |
| **Claude 3.5 Sonnet** | 复杂推理、代码生成 | $$$ |
| **GPT-4o** | 通用任务 | $$$ |
| **GLM-4** | 中文任务、性价比 | $$ |

### 3. 定期备份

```bash
# 导出所有会话
Settings → Export All → backup.tar.gz

# 建议每周备份一次
```

### 4. 监控使用量

```bash
# 查看使用统计
Settings → Usage

# 包含：
- Token 使用量
- API 调用次数
- 成本统计
```

### 1. 快捷键

| 快捷键 | 功能 |
|--------|------|

| `Cmd + N` | 新建会话 |
| `Cmd + /` | 显示帮助 |

模板示例：

帮我创建一个 FastAPI 项目，包含：
1. 用户认证
2. 数据库连接

上传多个文件 → Agent 自动分析 → 批量修改
