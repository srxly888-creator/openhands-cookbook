## 📖 Introduction

OpenHands Cookbook is a practical handbook for Chinese users to quickly get started with OpenHands - a powerful AI-driven development platform.

### ✨ Features

- 🇨🇳 **Chinese First** - All content in Chinese by default, lowering the learning curve
- 🚀 **Practical Oriented** - Provides real cases and best practices
- 📚 **Comprehensive** - Covers the complete path from beginner to master
- 🔄 **Continuously Updated** - Follows the latest OpenHands features

## 🎯 Quick Start

### Option 1: CLI Mode (Recommended for Beginners)

The simplest way to use, similar to Claude Code / Codex experience.

```bash
# Install OpenHands CLI
pip install openhands

# Start interactive session
openhands run

# Specify a task
openhands run "Help me create a FastAPI project"
```

**Detailed Guide**: [cookbook/cli-quickstart.md](cookbook/cli-quickstart.md)

### Option 2: Local GUI

Local visual interface, similar to Devin / Jules experience.

```bash
# Clone repository
git clone https://github.com/OpenHands/OpenHands.git
cd OpenHands

# Start service
make build
make run

# Access interface
open http://localhost:3000
```

**Detailed Guide**: [cookbook/gui-quickstart.md](cookbook/gui-quickstart.md)

### Option 3: Cloud (Free Trial)

No local deployment needed, use the hosted service directly.

1. Visit https://app.all-hands.dev
2. Login with GitHub / GitLab account
3. Select Minimax model (free)
4. Start using

**Detailed Guide**: [cookbook/cloud-quickstart.md](cookbook/cloud-quickstart.md)
